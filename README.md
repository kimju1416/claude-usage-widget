# Claude Usage Widget

Claude(claude.ai) 사용량을 항상 화면에 띄워두고 보는 Windows 데스크톱 위젯입니다.
코딩하다가 매번 claude.ai 설정 화면을 열어서 확인하는 게 번거로워서 만들었습니다.

- **5시간 세션 한도**, **주간 한도** 두 가지를 원형 그래프로 보여줍니다.
- **Fable 모델 주간 한도**는 트레이 툴팁에 항상 표시되고, 위젯 카드에는 옵션으로 켜고 끌 수 있습니다.
- **1분마다 자동 새로고침**됩니다.
- 화면에 항상 떠 있는 **위젯 카드** 모드와, 작업표시줄 **트레이 아이콘**만 쓰는 모드 중 선택할 수 있습니다.
- 별도 서버나 API 키 없이, claude.ai에 로그인한 세션으로 직접 값을 읽어옵니다.

## 화면 구성

### 위젯 카드

화면 한쪽에 항상 떠 있는 168×168px 크기의 작은 카드입니다.

- 왼쪽 원형 그래프: **5시간 세션** 사용률 (몇 시간/몇 분 후 재설정되는지 아래에 표시)
- 오른쪽 원형 그래프: **주간 한도** 사용률 (재설정 요일·시간 표시)
- 기본(컬러풀) 테마에서는 원마다 고유색이 있습니다: 5시간=청록, Fable=핑크, 주간=노랑. 튀는 색이 싫으면 트레이 메뉴에서 "차분한 톤(무채색)"으로 바꿀 수 있습니다
- 카드 아무 곳이나 **드래그**해서 원하는 위치로 옮길 수 있고, 위치는 자동 저장되어 다음 실행 때도 유지됩니다
- 우측 상단 새로고침(⟳) 버튼으로 즉시 다시 불러올 수 있습니다
- 맨 아래에 마지막 업데이트 시각이 표시됩니다
- 로그인이 필요하면 카드 안에 "로그인 필요 → 클릭" 버튼이 나타납니다
- 트레이 메뉴에서 "위젯에 Fable 표시"를 켜면 5시간/주간 원과 같은 크기의 세 번째 원(Fable 주간 사용률)이 가운데 추가되고, 카드 너비가 자동으로 넓어집니다

### 트레이 아이콘 모드

위젯 카드 대신 작업표시줄 트레이 아이콘 하나만 쓰고 싶다면 이 모드로 바꿀 수 있습니다. 아이콘에 마우스를 올리면 툴팁으로 "5시간: 30% (4시간 6분 후 재설정) / 주간: 84% (목 오후 8:59에 재설정) / Fable: 57% (목 오후 8:59에 재설정)" 형태로 숫자가 보입니다. Fable 항목은 위젯 카드 표시 옵션과 무관하게 트레이에는 항상 나옵니다.

## 설치

### 1) 설치 파일로 설치 (추천)

[Releases](../../releases)에서 `ClaudeUsageWidget-Setup-x.x.x.exe`를 받아 실행하면 자동으로 설치되고 바로 실행됩니다.

### 2) 소스에서 직접 빌드

```bash
git clone <이 저장소 주소>
cd ClaudeUsageWidget
npm install
npm run gen-assets   # 아이콘 생성
npm start            # 개발 모드로 실행
npm run dist          # Windows 설치 파일(.exe) 빌드 → dist/ 폴더
```

Node.js가 설치되어 있어야 합니다.

## 처음 실행 시

처음 실행하면 위젯에 "로그인이 필요해요"라고 표시됩니다.

1. 작업표시줄 트레이 아이콘을 **우클릭**
2. **"로그인 창 열기"** 클릭
3. 뜨는 창에서 claude.ai에 평소처럼 로그인 (Google/이메일/SSO 등)
4. 로그인이 완료되면 창이 자동으로 닫히고, 위젯에 실제 사용량이 표시됩니다

로그인 세션은 앱에 저장되기 때문에 이후에는 다시 로그인할 필요가 없습니다.
(단, claude.ai에서 로그아웃하거나 세션이 만료되면 다시 로그인해야 합니다.)

## 트레이 메뉴

작업표시줄 숨겨진 아이콘 화살표(`^`)를 눌러 크림색 별 모양 아이콘을 찾은 뒤 **우클릭**하면 다음 메뉴가 나옵니다.

| 메뉴 | 설명 |
|---|---|
| 위젯 카드로 보기 | 화면에 항상 떠 있는 카드 형태로 표시 (라디오 선택) |
| 트레이 아이콘으로만 보기 | 카드를 숨기고 트레이 아이콘 + 툴팁으로만 표시 (라디오 선택) |
| 위젯 투명도 | 위젯 카드의 불투명도를 100% / 85% / 70% / 55% 중에서 선택. 화면 작업을 가리지 않도록 흐리게 만들 수 있습니다 |
| 색상 테마 | 컬러풀(청록/핑크/노랑) 또는 차분한 톤(무채색) 중 선택 |
| 위젯에 Fable 표시 | 위젯 카드에 Fable 주간 사용률 원을 추가/제거 (트레이 툴팁에는 이 설정과 무관하게 항상 표시됨) |
| 지금 새로고침 | 기다리지 않고 즉시 최신 사용량을 다시 불러옴 |
| 로그인 창 열기 | claude.ai 로그인 창을 띄움. 로그인 완료가 감지되면 창이 자동으로 닫힙니다 |
| Windows 시작 시 자동 실행 | 켜두면 컴퓨터 부팅 시 위젯이 자동으로 실행됩니다 |
| 사용량 페이지 열기(브라우저) | claude.ai 사용량 설정 페이지를 기본 브라우저로 직접 엶 (위젯 값과 비교하고 싶을 때) |
| 디버그 로그 열기 | 값이 이상하거나 로그인이 안 될 때 확인할 수 있는 로그 파일(`%APPDATA%/ClaudeUsageWidget/debug.log`)을 엶 |
| 종료 | 앱을 완전히 종료 |

트레이 아이콘을 **왼쪽 클릭**하면(위젯 카드 모드일 때만) 카드를 보이기/숨기기 토글할 수 있습니다.

## 어떻게 동작하나요?

claude.ai에는 사용량을 조회하는 공개 API가 없어서, 로그인된 세션으로 `claude.ai`의 설정 &rarr; 사용량 페이지를 백그라운드 창에서 열어 화면에 표시된 텍스트("현재 세션 30% 사용됨", "모든 모델 84% 사용됨" 등)를 읽어와 그래프로 그려줍니다. 화면에는 보이지 않는 창에서 조용히 이루어지며, 1분마다 자동으로 반복됩니다.

이 방식의 특성상 claude.ai의 페이지 구성이 바뀌면 값을 못 읽어올 수 있습니다. 그런 경우 이슈로 알려주세요.

## 업데이트

새 버전 설치 파일을 다시 실행하면 기존 설치 위에 덮어써지고, **로그인 세션은 그대로 유지**됩니다. 다시 로그인할 필요 없습니다.

## 문제 해결

- **"로그인이 필요해요"가 계속 뜬다**: 트레이 우클릭 → "로그인 창 열기"로 다시 로그인해주세요. claude.ai 자체에서 로그아웃했거나 세션이 만료된 경우입니다.
- **숫자가 안 바뀐다 / 이상하다**: 트레이 우클릭 → "지금 새로고침"으로 즉시 갱신해보세요. 그래도 안 되면 "디버그 로그 열기"로 로그를 확인해주세요.
- **트레이 아이콘이 여러 개 뜬다**: v1.0.3부터 중복 실행 방지 기능이 있어 발생하지 않습니다. 이전 버전이라면 최신 버전으로 업데이트해주세요.

## 개인정보

- 로그인 정보(쿠키)는 이 앱의 로컬 프로필 폴더(`%APPDATA%/ClaudeUsageWidget`)에만 저장되며 외부로 전송되지 않습니다.
- claude.ai 서버 외에는 어떤 곳으로도 데이터를 보내지 않습니다.

Electron + 순수 HTML/CSS/JS. 별도 프레임워크 없이 가볍게 만들었습니다.
