# Claude Usage Widget

Claude(claude.ai) 사용량을 항상 화면에 띄워두고 보는 Windows 데스크톱 위젯입니다.
코딩하다가 매번 claude.ai 설정 화면을 열어서 확인하는 게 번거로워서 만들었습니다.

- **5시간 세션 한도**, **주간 한도** 두 가지를 원형 그래프로 보여줍니다.
- **1분마다 자동 새로고침**됩니다.
- 화면에 항상 떠 있는 **위젯 카드** 모드와, 작업표시줄 **트레이 아이콘**만 쓰는 모드 중 선택할 수 있습니다.
- 별도 서버나 API 키 없이, claude.ai에 로그인한 세션으로 직접 값을 읽어옵니다.

## 스크린샷

위젯 카드 모드는 화면 한쪽에 작은 카드로 떠서 드래그로 위치를 옮길 수 있습니다.
트레이 모드는 작업표시줄 아이콘에 마우스를 올리면 툴팁으로 퍼센트가 보입니다.

## 설치

### 1) 설치 파일로 설치 (추천)

[Releases](../../releases)에서 `ClaudeUsageWidget-Setup-x.x.x.exe`를 받아 실행하면 자동으로 설치되고 바로 실행됩니다.

### 2) 소스에서 직접 빌드

```bash
git clone <이 저장소 주소>
cd ClaudeUsageWidget
npm install
npm run gen-assets   # 아이콘 생성
npm start            # 개발 모드로 실행
npm run dist          # Windows 설치 파일(.exe) 빌드 → dist/ 폴더
```

Node.js가 설치되어 있어야 합니다.

## 처음 실행 시

처음 실행하면 위젯에 "로그인이 필요해요"라고 표시됩니다.

1. 작업표시줄 트레이 아이콘을 **우클릭**
2. **"로그인 창 열기"** 클릭
3. 뜨는 창에서 claude.ai에 평소처럼 로그인 (Google/이메일/SSO 등)
4. 로그인이 완료되면 창이 자동으로 닫히고, 위젯에 실제 사용량이 표시됩니다

로그인 세션은 앱에 저장되기 때문에 이후에는 다시 로그인할 필요가 없습니다.
(단, claude.ai에서 로그아웃하거나 세션이 만료되면 다시 로그인해야 합니다.)

## 트레이 메뉴

작업표시줄 아이콘을 우클릭하면 다음 메뉴가 있습니다.

| 메뉴 | 설명 |
|---|---|
| 위젯 카드로 보기 | 화면에 항상 떠 있는 카드 형태로 표시 |
| 트레이 아이콘으로만 보기 | 카드를 숨기고 트레이 아이콘 + 툴팁으로만 표시 |
| 지금 새로고침 | 즉시 최신 사용량을 다시 불러옴 |
| 로그인 창 열기 | claude.ai 로그인 창을 띄움 |
| Windows 시작 시 자동 실행 | 켜두면 부팅 시 자동으로 위젯이 실행됨 |
| 사용량 페이지 열기(브라우저) | claude.ai 사용량 설정 페이지를 기본 브라우저로 엶 |
| 디버그 로그 열기 | 문제가 있을 때 확인할 수 있는 로그 파일을 엶 |
| 종료 | 앱 완전 종료 |

## 어떻게 동작하나요?

claude.ai에는 사용량을 조회하는 공개 API가 없어서, 로그인된 세션으로 `claude.ai`의 설정 &rarr; 사용량 페이지를 백그라운드 창에서 열어 화면에 표시된 텍스트("현재 세션 30% 사용됨", "모든 모델 84% 사용됨" 등)를 읽어와 그래프로 그려줍니다. 화면에는 보이지 않는 창에서 조용히 이루어지며, 1분마다 자동으로 반복됩니다.

이 방식의 특성상 claude.ai의 페이지 구성이 바뀌면 값을 못 읽어올 수 있습니다. 그런 경우 이슈로 알려주세요.

## 개인정보

- 로그인 정보(쿠키)는 이 앱의 로컬 프로필 폴더(`%APPDATA%/ClaudeUsageWidget`)에만 저장되며 외부로 전송되지 않습니다.
- claude.ai 서버 외에는 어떤 곳으로도 데이터를 보내지 않습니다.

Electron + 순수 HTML/CSS/JS. 별도 프레임워크 없이 가볍게 만들었습니다.
